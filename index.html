The 5-Layer Architecture

Replacing manual parking supervision with an end-to-end automated machine learning pipeline.

1. Video Input Layer

Continuous live CCTV/Webcam feed captured. Frames are extracted and processed live via OpenCV.

2. AI Detection Layer

Computer vision pipeline analyzes frames. Vehicles are detected and mapped to specific parking slot coordinates.

3. Analytics Engine

Calculates total, occupied, and available slots in real-time. Computes overall occupancy rate percentages.

4. Flask Backend

Serves as the bridge. Streams processed video and exposes the Analytics API.

GET /video_feed → Live Stream
GET /api/stats → Real-time JSON Data

5. React Dashboard Layer

Modern interactive UI built with TypeScript and Tailwind CSS. Features live CCTV rendering, animating analytics cards, and AI status indicators.
